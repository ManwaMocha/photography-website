# Mocha Gallery: Travel Photography Website

## Author
Augustus Mocha

## Description
Mocha Gallery is a simple travel photography website that showcases different destinations from around the world. The website displays images in a clean grid layout, with each image presented as a card containing the place name, location, and a short description. It is designed to be visually appealing, easy to navigate, and user-friendly.
